Final Project README - Daniel Kantor

1. Put webCrawler.py and calculateRanking.py in the same folder
2. Run in command line python -m nltk.downloader stopwords to download the list of stopwords
3. Run pip install ntlk
4. Run webCrawler.py (change the values of the two booleans to determine which optimizations you want)
5. After running webCrawler.py run calculateRankings.py (make sure to only select optimizations for which you have run webCrawler.py with)
